# Cortex

The AI operating system for companies.

## Structure

- `backend/` — Node.js + TypeScript (Express)
- `frontend/` — React + TypeScript + Tailwind CSS (Vite)

## Getting started

### Backend

```bash
cd backend
npm install
npm run dev
```

Runs on http://localhost:3001. Health check: `GET /health`.

### Frontend

```bash
cd frontend
npm install
npm run dev
```

Runs on http://localhost:5173.
